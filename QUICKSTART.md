# Быстрый старт

## Установка зависимостей

```bash
npm install
```

## Запуск на Android

1. Убедитесь, что у вас запущен эмулятор Android или подключено физическое устройство
2. Запустите Metro bundler:
   ```bash
   npm start
   ```
3. В другом терминале запустите приложение:
   ```bash
   npm run android
   ```

## Первый запуск

При первом запуске приложение запросит разрешения:
- **Камера** - для съемки видео и фото
- **Микрофон** - для записи звука в видео
- **Хранилище** - для сохранения файлов

Разрешите все запрошенные разрешения.

## Тестирование функций

### Протокол с видео
1. Нажмите "Протокол с видео"
2. Нажмите красную кнопку для начала записи
3. Нажмите кнопку "Стоп" для завершения
4. Нажмите "Отправить"

### Протокол с фото
1. Нажмите "Протокол с фото"
2. Нажимайте "Добавить фото" для съемки
3. Нажмите "Отправить" когда готово

### Протокол со скриншотами
1. Нажмите "Протокол со скриншотами"
2. Используйте телефон в обычном режиме
3. Нажимайте на плавающую синюю кнопку справа для создания скриншотов
4. Нажмите "Завершить" для отправки

### Просмотр протоколов
1. Нажмите "Список протоколов" на главном экране
2. Выберите протокол для просмотра
3. Нажмите "Открыть PDF"

## Важные замечания

- API интеграция является моковой - для подключения реального API отредактируйте `src/services/ApiService.js`
- Все файлы сохраняются в папке приложения на телефоне
- Уникальный идентификатор устройства генерируется автоматически
